🌱 Funcionalidades

Todo lo que necesitas para tu invernadero

Desde sensores hasta reportes automatizados, EchoSmart cubre cada aspecto del monitoreo ambiental en invernaderos.

🌡️

Monitoreo en Tiempo Real

Lectura continua de 5 variables ambientales con actualización instantánea vía WebSocket cada segundo.

🚨

Alertas Configurables

Reglas flexibles con evaluación local y en la nube. Notificaciones por email, SMS, push y webhooks.

📊

Reportes Automatizados

Generación de reportes PDF y Excel con gráficas embebidas. Programación diaria, semanal o personalizada.

🔧

Edge Computing

El gateway opera de forma autónoma con caché local, garantizando continuidad ante pérdida de conectividad.

🌿

Control de Actuadores

Encendido/apagado remoto de riego, ventilación e iluminación para automatizar el clima del invernadero.

🤖

Analítica Predictiva

Modelos de ML para predicción de temperatura, detección de anomalías y optimización del cultivo.

🏗️ Arquitectura

Arquitectura Edge-to-Cloud

Tres capas diseñadas para operar de forma autónoma y escalable en entornos agrícolas.

Edge

🔧 Gateway

Raspberry Pi 4B con 5 drivers de sensores, MQTT local y evaluación de alertas.

  • Python 3.9+ · SQLite
  • Mosquitto MQTT 2.0+
  • HAL (GPIO, I2C, 1-Wire)
  • SSDP Auto-Discovery
  • Operación offline autónoma
Cloud

☁️ Backend

API REST con microservicios, almacenamiento dual y procesamiento asíncrono.

  • FastAPI · JWT + RBAC
  • PostgreSQL 14+ · InfluxDB
  • Redis 7+ · RabbitMQ
  • Docker · Kubernetes
  • WebSocket tiempo real
Cliente

📱 Frontend

Dashboard web y app móvil con gráficas en tiempo real y gestión completa.

  • React 18+ · Redux Toolkit
  • Recharts · Material-UI
  • React Native (Expo)
  • i18n multilenguaje
  • PWA offline support
📋 Normativas

Normas y Estándares Aplicables

EchoSmart cumple con normas nacionales e internacionales para agricultura protegida e IoT.

🇲🇽

NOM-001-SAGARPA

Norma Oficial Mexicana para agricultura protegida y producción en ambientes controlados.

🌐

ISO/IEC 30141

Arquitectura de referencia para Internet de las Cosas (IoT). Marco de diseño de sistemas IoT.

🔒

ISO 27001 / IEC 62443

Seguridad de la información y ciberseguridad en sistemas de automatización industrial.

📊

NMX-I-27001-NYCE

Adaptación mexicana de ISO 27001 para gestión de la seguridad de la información.
